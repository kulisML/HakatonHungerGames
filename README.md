# Agent World

Веб‑приложение с “живущими” AI‑агентами на карте: они общаются, спорят, воюют, помнят прошлое и меняют отношения.

## Быстрый старт

1) Создай `.env` рядом с `.env.example` и вставь ключи OpenRouter:

```
OPENROUTER_KEYS=key1,key2,key3,key4,key5
```

2) Установи зависимости и запусти dev:

```
npm install
npm run dev
```

По умолчанию:
- API: http://localhost:8787
- UI: http://localhost:5173
